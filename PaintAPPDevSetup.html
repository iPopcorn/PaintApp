PaintAPP Dev Setup

1. Download and install PyCharm Community: https://www.jetbrains.com/pycharm/download/#section=windows

2. Download and install Python 3.7 (make sure to include pip): make sure you know where it installs to, you will need it for PyCharm. https://www.python.org/downloads/release/python-371/

3. Install the Kivy Framework: https://kivy.org/doc/stable/installation/installation-windows.html#install-win-dist

4. Download git: https://git-scm.com/download/win

5. Install git > on the 5th screen, select 'git from command line and also 3rd party software' > keep all other options default and install

6. Run PyCharm > Select 'check out from version control' > git

7. Enter URL: https://github.com/iPopcorn/PaintApp.git

8. Click 'clone'

9. When PyCharm opens, go to File > Settings

10. On the left look for Project > Project Interpreter.

11. Add a new interpreter to the project by clicking the gear icon to the right of the textbox.

12. Select System Interpreter on the left

13. Choose the path to your Python installation

14. Click OK, apply all changes and close settings

15. In the top right, next to the play button icon click the dropdown and select edit configurations

16. Click the '+' icon in the top left to add a new configuration

17. Select Python

18. Change the script path to main.py

19. Set Python Interpreter to Python 3.7

20. Apply changes and click OK to close the window

21. Select the configuration you added and click the play button
